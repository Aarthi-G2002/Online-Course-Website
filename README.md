# Online Course Website

### This project is a modern and responsive online learning platform built using React and Vite. It allows users to explore different courses, learn about services, and contact the platform easily. The website includes multiple pages such as Home, About, Services, Courses, and Contact, all designed with a clean and professional layout. Smooth navigation and engaging visuals make it user-friendly on all devices.

 # Tech Stack:
 ## React, Vite, React Router DOM, CSS

 # Features: 
 ## Multi-page navigation, responsive design, interactive course sections, simple UI, and easy deployment on Netlify.
